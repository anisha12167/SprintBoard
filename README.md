# SprintBoard

SprintBoard is a task management web application inspired by agile workflow systems. It helps users organize projects using boards, columns, and task cards.

## Features
- Drag and drop task management
- Create and organize workflow boards
- Column based task tracking
- Responsive UI
- Real-time task organization

## Tech Stack
- React.js
- JavaScript
- Vite
- Tailwind CSS
- Context API

## Future Improvements
- Task priority
- Due date support
- Authentication system
